## Overview

The Purchase Order Management System is a web-based procurement automation platform designed to streamline the workflow from Request for Quotation (RFQ) to Purchase Order generation and approval.

The system improves procurement efficiency, transparency, and accountability through role-based access control and automated workflows.

---

## Features

* User Authentication using JWT
* Role-Based Access Control
* Request for Quotation (RFQ) Management
* Vendor Management
* Purchase Order Generation
* Approval Workflow System
* Dashboard and Reporting
* Secure Database Integration
